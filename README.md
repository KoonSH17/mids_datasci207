# Machine Learning Fundamentals

A collection of hands-on notebooks covering supervised learning fundamentals,
implemented in Python using TensorFlow and scikit-learn.

---

## Notebooks

| # | Topic | Key Concepts |
|---|-------|-------------|
| 01 | Introduction | Python for ML, data exploration, preprocessing |
| 02 | Linear Regression | Ordinary least squares, gradient descent, evaluation metrics |
| 03 | Linear Regression with TensorFlow | Implementing regression in TensorFlow, model training loops |
| 04 | Logistic Regression with TensorFlow | Binary classification, sigmoid, cross-entropy loss |
| 05 | Multiclass Classification | Softmax, one-vs-rest, confusion matrix |
| 06 | Neural Networks | Feedforward networks, backpropagation, Keras Sequential API |
| 07 | Decision Forests | Random forests, feature importance, land cover classification (EuroSAT dataset) |

---

## Stack
- Python 3
- TensorFlow / Keras
- scikit-learn
- pandas, NumPy, Matplotlib
